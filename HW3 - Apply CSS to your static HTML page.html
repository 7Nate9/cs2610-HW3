Nate's Git Cheat Sheet

Welcome to my Git cheat sheet! Here you'll find a list of helpful git commands, their descriptions, and working examples.
Alternatively, see the full documentation for git.

List of Commands

Command Description Example
git init Creates a git repository in the current directory. git init
git add Displays the sum of all files in the repository. git add
git commit Saves changes to the local repository. (May prompt awkward conversation if repository isn't ready for that kind of thing.) git commit -m "I'm ready to take this to the next level."
git push Pushes saved changes down the tubes to a remote repository. If tubes are clogged, apply pipe snake as needed. git push really hard
git go back The git equivalent of ctrl + Z. Undoes whatever you screwed up. git go back!!1!
git woo Reassures repository after a failed commit. git woo -m "I'm a different person now."
git django Releases a python.
git freaky Gives the repository a bottle of bourbon and tells it to have a good time. git freaky, git freaky, everybody in the house git freaky.
git r'lyeh Awakens the Great Old Ones from their eternal slumber. Launches the earth into an unending age of blood and darkness. May corrupt repository.
